# Stable Diffusion Text-to-Image Generator

This project generates images from text prompts using Stable Diffusion.

## Technologies Used
- Python
- Hugging Face Diffusers
- Stable Diffusion XL
- Gradio
- Google Colab

## How It Works
1. User enters a text prompt.
2. Stable Diffusion model processes the prompt.
3. Diffusion model generates an image.
4. Image is displayed using a Gradio interface.

Example Prompt:
teddy bear on tree
